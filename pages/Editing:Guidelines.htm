Starting new pages

When creating a new page, to have that nice box at the top right of the page you're editing, use the following template and just add in the necessary information: 

{{app
|appname=
|imagename=
|platform=
|download=
|developer=
|version=
|status=
|homepage= (insert url)
|homepage_name= (insert hyperlinked text)}}

This will render a box like this:

'
[[Image:|320px]]
Platform(s):
Download(s):
Developer(s):
Version:
Status:
Homepage: [(insert url) (insert hyperlinked text)]


Download Links

All file downloads should be hosted on the Dingoo File Archive. If the emulator, game, app, etc. is not stored there, upload the file to the Dingoo File Archive, and link to the resulting download page. Do NOT link directly to the file itself, but to the summary page for the download. This way, the downloader will be able to see the description for what they're downloading on the Archive page. The only other acceptable download links would be links to project pages, hosted at sites like github, SourceForge, and Google Code. Download links on sites like mediafire, filedropper, or forum download attachments are unacceptable.

Controls sections

Use the following wiki code in order to place controls instructions in articles: 

{| class="wikitable" style="left;text-align:center;"
|-
| {{Up}} || Up
|-
| {{LeftRight}} || Movement
|-
| {{Down}} || Down
|-
| {{Start}} || Start
|-
| {{Select}} || Select
|-
| {{L}} || L
|-
| {{R}} || R
|-
| {{A}} || A
|-
| {{B}} || B
|-
| {{X}} || X
|-
| {{Y}} || Y
|-
|}

This will render the controls like this:

Template:Up Up
Template:LeftRight Movement
Template:Down Down
Template:Start Start
Template:Select Select
Template:L L
Template:R R
Template:A A
Template:B B
Template:X X
Template:Y Y

Grammar

Please do recheck what you have written down for grammar and spelling errors. Copying and pasting your text into a word processor for spellcheck may help.
