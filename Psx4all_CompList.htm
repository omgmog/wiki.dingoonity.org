Few tips to get the best performance

  • Test the compressed/uncompressed rom. It might effect the performance
  • Always test the game with, and without bios. The performance difference might be noticeable (even +50%)
  • Overclock to 430MHz
  • Interlace increases the framerate
  • Experiment with frameskip. Usually 2/3 gives the best results
  • For some games setting "Cycle Multiplier" to greater values gives a performance boost
  • Use Wall Clock Timing for slow games, it works like auto frameskip most of the time, but can be buggy or not work
